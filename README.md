# files-api 文件微服务

> 用于处理文件上传下载到服务器的功能

## Build Setup

``` bash
# 下载 dependencies
npm install

# 启动项目(本地版)
npm run start
# 启动项目(阿里云OSS版)
npm run oss
```
## 配置
> nodeapi\files-api\config\dev.yml 用于设置系统配置

## 接口配置
> nodeapi\files-api\config\文件上传微服务接口.postman_collection.json 用于导入postman生成接口

### 文件上传接口
